# Microwave Simulation

This project uses C and Assembly to program a Freescale HCS12 Dragonboard Microcontroller to simulate functions of a microwave.

Components Used:
• LCD
• Hex Keypad
• LEDS
• DIP Switch
• Potentiometer
• Thermometer
• SCI
• Timer module
• Speaker
• DC Motor
• Servo Motor
• External LED
• External Potentiometer
• External Breadboard
